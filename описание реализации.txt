Для работы с файлами создан класс "Wrapper", в котором хранятся имя файла, путь до него,
количество его включений в другие файлы и ссылки на его зависимости ("Wrapper").
При работе программы функция "finder_source_cod" проходит по заданной директории и создаёт
массив  "Wrapper" для всех файлов с разрешением ".cpp". Далее функция "finder_in_file" для 
найденных файлов ищет имена зависимостей и добавляет их в массив "Wrapper".Для поиска одного файла
используется рекурсивная функция "finder_headers". Чтобы найти все зависимости исходных файлов,
вызывается функция "analyzer", которая для каждого "Wrapper" из сохранённого массива вызывает
"finder_headers" и, если находит, применяет "finder_in_file". Далее процесс рекурсивно повторяется.
Чтобы избежать циклов, используется ещё один массив, в котором хранятся уже найденные файлы.
В итоге, мы получаем массив "Wrapper" исходных файлов, для каждого из которых сохранён
массив "Wrapper" зависимостей, указывающих на свои массивы "Wrapper" и т.д..
